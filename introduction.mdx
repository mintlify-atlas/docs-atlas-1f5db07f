## Welcome to Agrobotics FRC Robot

This is the official documentation for the Agrobotics FIRST Robotics Competition (FRC) robot codebase. Built with WPILib 2020 and Java 11, this robot features a sophisticated drivetrain system with path recording and autonomous replay capabilities.

## Key Features

<CardGroup cols={2}>
  <Card title="Command-Based Architecture" icon="code">
    Built on WPILib's command-based framework for modular, maintainable robot code
  </Card>
  
  <Card title="TitanQuad Motor Control" icon="gear">

  <Card title="Dual Drive Modes" icon="gamepad">

  <Card title="Multi-Path Storage" icon="database">

  </Card>

## Project Structure

The robot code is organized following WPILib best practices:

```
src/main/java/frc/robot/
├── Robot.java              # Main robot class
├── RobotContainer.java     # Subsystem and command initialization
├── Main.java              # Application entry point
├── Constants.java         # Hardware IDs and configuration constants
├── subsystems/
│   └── DriveTrain.java    # Drivetrain subsystem
├── commands/
│   └── Teleop.java        # Teleoperated drive command
└── gamepad/
    └── OI.java            # Operator interface
```

## Hardware Overview

- **Motor Controllers**: 4x Studica TitanQuad (ID: 42)
- **Gyroscope**: Kauai Labs NavX (SPI MXP port)
- **Encoders**: TitanQuad integrated encoders with distance calculation
- **Control**: Xbox-style gamepad for driver input
